Hovercraft

Hovercraft is a JQuery library of web hover effects, built for use on buttons, links, text, and more. It includes markup, code, and style for simple effects, and also pre-styled elements and links for drop-down menus and social media. Hovercraft was inspired by my frequent need of hover effects, and how little I enjoyed re-typing the code. Thus, everytime I create something new, it will be added to this library for all to enjoy!
